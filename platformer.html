Python / Pygame

Knight in the Night
at Night.

A 2D platformer adventure filled with action, style, and challenge. Built in Python from scratch.

Play Now → ← All Projects

01 / character

Unique Character Design

The main character, a valiant knight, is animated with a full sprite sheet including idle, run, jump, and multiple attack moves. Special effects like landing dust puffs and smoke trails enhance gameplay immersion.

02 / combat

Dynamic Enemy AI

Enemies like Orc Warriors and Skeleton Archers use state-based AI for patrolling, searching, and attacking. The archer features a vector-based ranged attack system, while all combat interactions show floating damage text for real-time feedback.

03 / inventory

Inventory and Item System

A Diablo-style inventory lets players drag and equip items with tooltips displaying rarity and stats. Equipped gear directly affects the knight's attributes, offering tactical flexibility in combat and exploration.

04 / progression

Progression System

Defeating enemies rewards XP, which can be used to level up and gain attribute points. Players can invest in health, stamina, attack power, and more to customize their build as they progress.

05 / skills

Skill Tree

As players level up, they unlock skill points to spend on a structured skill tree. Each passive skill enhances attributes further, offering bonuses like dodge chance, lifesteal, XP gain, and critical damage multipliers.

06 / economy

Shop System

Enemies drop gold that can be spent at in-game shops to purchase new gear. Items come with detailed stat bonuses, and players can also sell unwanted equipment for half its value, creating a rewarding in-game economy.

07 / world

Handcrafted Maps with Tiled

Levels built using Tiled, supporting layered environments, animated tiles, and interactive objects like save points. Parallax backgrounds and smooth camera tracking create a rich and immersive world for players to explore.
